üìã How to Embed the Booking Widget

There are two ways to embed our booking widget on your website. Both are simple and work on any website:

Option 1: Simple Script Tag (Recommended)

Best for: Easy setup, works everywhere

Add this code to your website where you want the booking widget to appear:

<!-- CRM Booking Widget --> <div id="crm-booking-widget-container"></div> <script src="https://crm-pi-eight.vercel.app/api/widget?tenant=YOUR_BUSINESS_ID"></script>

Replace "YOUR_BUSINESS_ID" with your actual business ID (e.g., "demo")

Option 2: Manual Initialization

Best for: Advanced users who want more control

<!-- Place this where you want the widget --> <div id="my-booking-widget"></div> <!-- Load the widget script --> <script src="https://crm-pi-eight.vercel.app/api/widget?tenant=YOUR_BUSINESS_ID"></script> <!-- Initialize with custom container ID --> <script> // Wait for the widget to load, then initialize setTimeout(function() { if (window.CRMBookingWidget) { window.CRMBookingWidget.init('my-booking-widget'); } }, 100); </script>

üîß Business ID Setup

Each business customer gets their own unique business ID. For example:

  • Demo Business: tenant=demo
  • Your Business: tenant=your-business-id

You can find your business ID in your CRM dashboard.
